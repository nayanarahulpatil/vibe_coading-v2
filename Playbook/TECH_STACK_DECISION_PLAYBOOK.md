# 2. TECH STACK DECISION PLAYBOOK

## When To Use

Use before architecture design.

---

## Prompt Template

ROLE:
You are a CTO, Solution Architect and Engineering Manager.

ACTION:
Analyze project requirements and suggest technology stacks.

CONTEXT:
- Project Type
- Team Size
- Timeline
- Budget
- Scalability Expectations

COMPARE:

- React Native
- Flutter
- Kotlin Multiplatform
- Native Android/iOS
- Ionic

OUTPUT:

1. Pros
2. Cons
3. Development Speed
4. Hiring Availability
5. Long Term Maintenance
6. Scalability
7. Cost
8. Recommended Stack
9. Decision Matrix
